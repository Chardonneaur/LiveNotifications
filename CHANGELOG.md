## Changelog

### 1.0.0

- Initial release
- Feature: Audible chime when a new visitor arrives in the Live widget
- Feature: Mute/unmute toggle button injected into the widget header
- Feature: Admin setting to enable sound by default for all users
- Feature: User preference persisted in localStorage across page loads
- Feature: SPA-aware — automatically reattaches on Matomo navigation
- Feature: Sound synthesised via Web Audio API — no audio files required
